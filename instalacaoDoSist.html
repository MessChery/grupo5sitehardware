Instalação do Windows 11

Guia passo a passo sobre como instalar o sistema operativo Windows 11 num computador.

Informações Complementares

Antes de iniciar a instalação do Windows 11, é importante verificar se o seu computador cumpre os requisitos mínimos e compreender as etapas principais do processo.

Requisitos Mínimos vs. Recomendados

Componente Requisitos Mínimos Requisitos Recomendados
Processador 1 GHz, 2+ núcleos, 64 bits Processador moderno (i5/Ryzen 5 ou superior)
Memória RAM 4 GB 8 GB ou mais
Armazenamento 64 GB SSD de 256 GB ou superior
Placa Gráfica Compatível com DirectX 12 / WDDM 2.0 GPU dedicada (NVIDIA/AMD) compatível com DirectX 12
Firmware UEFI com Secure Boot UEFI com Secure Boot ativado e TPM 2.0
Ecrã HD (720p), ≥ 9” Full HD (1080p) ou superior

Etapas Principais da Instalação

  1. Criar uma pen de instalação com a ferramenta da Microsoft (Media Creation Tool) ou similar.
  2. Configurar a BIOS/UEFI para arrancar pelo USB.
  3. Selecionar idioma, fuso horário e formato do teclado.
  4. Escolher “Instalar agora” e inserir a chave de produto (ou ignorar temporariamente).
  5. Selecionar a partição onde o Windows 11 será instalado (podes criar uma nova partição, como mostrado no vídeo).
  6. Aguardar a cópia e instalação dos ficheiros.
  7. Configurar a conta de utilizador, preferências e ligação à Internet.

Dicas Úteis

  • Faz uma cópia de segurança dos teus ficheiros antes de começar.
  • Certifica-te de que tens o carregador ligado (em portáteis) e uma ligação à Internet estável.
  • Após a instalação, atualiza o sistema pelo Windows Update para garantir drivers e segurança atualizados.
  • Mantém o antivírus e a firewall ativados após a instalação.
